Recipe Finder is a user-friendly web application that allows you to search for recipes by name and view detailed information including ingredients and instructions. The app uses **TheMealDB API** to fetch real-time recipe data.

Features
- Search recipes by name
- View recipe details with ingredients and instructions
- Interactive recipe cards with **View Recipe** popup
- Clean, responsive design

Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- TheMealDB API
